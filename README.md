# What2Wear

What2Wear is a simple web application that suggests what to wear based on real-time weather data.

It fetches live weather information using the OpenWeatherMap API and dynamically recommends outfits depending on temperature and weather conditions.

---

##  Features

-  Search weather by city
-  Real-time temperature display
-  Dynamic weather icon updates
-  Outfit suggestions based on temperature
-  Rain detection with umbrella suggestion
-  Responsive design
-  Clean and minimal UI

---

##  Built With

- HTML
- CSS
- JavaScript (DOM Manipulation)
- OpenWeatherMap API

---

##  How It Works

1. User enters a city name
2. App fetches weather data using OpenWeather API
3. Temperature and weather condition are extracted
4. UI updates dynamically
5. Outfit suggestion changes based on:
   - Temperature range
   - Rain conditions
